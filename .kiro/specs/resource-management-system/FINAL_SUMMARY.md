### 1. 递归树形渲染

```typescript
function convertToTreeData(nodes: any[]): ResourceTreeNode[] {
  return nodes
    .sort((a, b) => (a.sortOrder || 0) - (b.sortOrder || 0))
    .map(node => ({
      key: node.id.toString(),
      children: node.children ? convertToTreeData(node.children) : undefined,
    }))
}
```
