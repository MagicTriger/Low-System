# API 变更日志

## v1.0.0 (2024-01-01)

### 新增功能

- ✨ 初始版本发布
- ✨ 认证与授权API
  - 用户登录
  - 用户登出
  - Token刷新
- ✨ 菜单管理API
  - 获取菜单树
  - 创建菜单
  - 更新菜单
  - 删除菜单
- ✨ 资源管理API
  - 获取资源列表
  - 创建资源
  - 更新资源
  - 删除资源
  - 获取资源详情

### 技术规范

- OpenAPI 3.0.3
- Bearer Token认证
- RESTful API设计
- 统一响应格式

### 限流策略

- 登录接口: 5次/分钟
- 文件上传: 10次/分钟
- 其他接口: 100次/分钟
